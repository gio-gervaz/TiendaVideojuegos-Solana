# TiendaVideojuegos-Solana

Proyecto de smart contract desarrollado en Solana utilizando el framework Anchor.

## Funcionalidades

- Crear tienda de videojuegos
- Agregar juegos al catálogo
- Eliminar juegos

## Estructura del proyecto

programs/
Contiene el smart contract en Rust

tests/
Contiene pruebas del programa

## Tecnologías utilizadas

- Solana
- Rust
- Anchor Framework
